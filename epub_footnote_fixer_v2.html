About this tool
🐛
The bug
KF8 Joint MOBI files store footnote anchors in a way that shifts them by one when converted to EPUB. Tapping note 2 jumps to note 1, note 3 to note 2, and so on.
🔧
The fix
The tool rewires every footnote link to point to the correct anchor, and optionally replaces verbose back-link text with a clean ↩ symbol.
📱
Who is this for
Boox e-reader users (NeoReader) who converted Kindle books to EPUB and found footnote navigation broken.
🔒
Privacy
Your file never leaves your device. Everything runs locally in the browser — no server, no upload, no tracking.
For best results, first unpack your MOBI using KindleUnpack GUI to get the clean KF8 source, then zip the mobi8 folder into an EPUB before using this tool. This ensures the table of contents is also correct.
